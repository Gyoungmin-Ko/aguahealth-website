# 아그와헬스 Pricing 제안서 — 구매로 이어지는 가치·심리 설계

**목표:** pricing test의 가치 중심 모델 + 심리 원칙을 결합해, **진실만 사용**하면서도 고객이 “이건 사야 한다”고 느끼게 하는 구조와 문구를 제안합니다.

---

## 1. 전체 구조 요약

| 구분 | 내용 | 심리 원칙 |
|------|------|------------|
| **원타임** | 드롭다운으로 모듈·가격 선택 → 즉시 결제 화면 | 손실 회피(한 건만 시도), 호혜(무료 견적 먼저) |
| **원타임 묶음** | 여러 모듈 선택 시 단품가 표시 + 합산 → 결제 | 단위 편향, 대비 효과(묶음 할인 인식) |
| **구독 3단계** | Starter 50만원 / **Professional 120만원** / Enterprise 문의 | Good–Better–Best, 미끼, 중앙 무대 |
| **ROI 계산기** | 시간 절감·기회비용·이익 상승 시뮬레이션 | 손실 회피→이득 프레이밍, 숫자 앵커 |
| **Expert Review** | 신뢰를 더하는 전문가 검수 옵션 | 권위 편향, 후광 효과 |

---

## 2. 원타임 단품 — “한 건만 해도 결제하고 싶게”

### 2.1 가격 테이블 (pricing test 기준, 진실만)

| 가격대 | 모듈 (동일 가격) |
|--------|------------------|
| **10만원** | 예상 보험가격 산정, 기존기술 여부 평가, 선진입 의료기술 평가, 신의료기술평가 준비, 행위·치료재료 등재 전략 |
| **100만원** | 외국 현황 리포트, 교과서·의약품집 수재, 학술지 수재, 예상 대체가능약제, 건강보험 등재 전략 수립, 약가협상 전략, 위험분담제 시뮬레이션, 퇴장방지의약품·원가보전, 재정영향분석 |
| **500만원** | 체계적 문헌고찰(SLR), 간접비교 시뮬레이터 |
| **1,000만원** | 경제성평가 |

### 2.2 UI·플로우 (심리 반영)

- **드롭다운**: “모듈 선택” → 선택 시 **단품 가격 즉시 표시** (인식 우선, 투명성).
- **앵커**: 상단에 “글로벌 마켓액세스 컨설팅은 프로젝트당 수천만원대가 일반적입니다.” (사실일 때만) → 10만원·100만원이 “저렴”하게 인식되도록.
- **호혜**: “먼저 무료로 견적만 받아보세요. 48시간 내 제안.” → 문의 부담을 낮추고, 견적 후 결제로 자연스럽게 연결.
- **탈출 경로**: “견적 후 진행 여부는 귀사가 결정합니다. 별도 의무 없습니다.”로 불안 해소.

### 2.3 카피 (진실만)

- **10만원대**: “20시간 걸리던 기초 조사를 10분으로. 한 건만 써보고 결정하세요.”
- **100만원대**: “등재 전략·외국 현황·원가보전까지. 1건당 100만원, 프로젝트 단위 결재에 맞춤.”
- **500만원·1,000만원**: “고난도 분석(SLR·경제성평가)은 전문 컨설팅 대비 합리적인 단가로.”

---

## 3. 원타임 묶음 — “여러 개 고르면 더 뚜렷한 가치”

- **드롭다운 복수 선택** → 선택한 모듈별 **단품 가격**을 나열하고 **합계**를 실시간 표시.
- **옵션**: “N개 이상 선택 시 00% 할인”을 **실제로 적용할 때만** 표기 (거짓 결핍·거짓 할인 금지).
- **심리**: 단위 편향(한 건 단위로 인식), 대비 효과(여러 건 묶음이면 “한 건당 체감 단가”가 낮아 보임).

---

## 4. 구독 3단계 — “Professional을 선택하게”

### 4.1 플랜 정의 (pricing test + 심리)

| 플랜 | 가격 | 크레딧 | 핵심 메시지 | 심리 |
|------|------|--------|-------------|------|
| **Starter** | 월 50만원 | 10 Credits | 반복 업무 제로, 실무 가속화 | 효율·손실 회피(시간 절감) |
| **Professional** | 월 120만원 | 40 Credits | 성공적인 등재와 이익 극대화 | **중앙 무대**, 후광, 이득 프레이밍 |
| **Enterprise** | 별도 문의 | Custom/무제한 | 독보적인 기술력의 전사 도입 | 미끼(고가 앵커), 권위 |

### 4.2 Professional을 “구매할 수밖에 없는” 위치로

- **뱃지**: “성공 확률 극대화” 또는 “가장 많이 선택” (실제 데이터 있을 때만 “가장 많이 선택” 사용).
- **시각**: 카드 테두리·배경으로 Professional만 강조 (중앙 무대 효과).
- **카피 (진실만)**:
  - “0.1%의 약가 차이가 가져올 수십억 원의 추가 이익을 고려하면, 120만 원은 매우 낮은 투자입니다.”
  - “외국 현황·위험분담제 시나리오를 미리 검토해 등재 실패 리스크를 사전에 줄입니다.”
- **비교표**: 포함 모듈 → 크레딧 → 지원 수준 → 가격 순으로 한눈에 비교 (점진적 노출, 인식 우선).

### 4.3 Starter·Enterprise 역할

- **Starter**: “한 건만 써보고 싶다”는 진입용. 50만 원이 “시도 비용”으로 느껴지게 (앵커: 120만 원이 있으므로 50만 원은 낮게).
- **Enterprise**: “우리는 더 많이 쓴다”는 고객용. Custom을 두어 120만 원이 “적당한 중간”으로 보이게 (미끼 효과).

---

## 5. ROI 계산기 — “숫자로 구매 이유를 만들어 주기”

### 5.1 이름·위치

- **이름**: “업무 생산성 및 이익 기회 계산기” (가치 프레이밍).
- **위치**: 구독 플랜 **위** 또는 **옆** — 계산 결과를 본 뒤 플랜을 고르게.

### 5.2 입력·출력 (진실만)

**입력**
- 한 달에 검토하는 제품/적응증 수
- 담당자 시간당 인건비 (또는 연봉/연간 근무시간)
- 현재 1건당 업무 소요 시간(시간)

**출력**
1. **시간 절감**: 아그와헬스 도입 시 매달 아낄 수 있는 **N시간** (실제 모델에 맞게 산출).
2. **기회 가치**: 그 시간을 전략 업무에 쓸 때의 **잠재적 부가가치** (보수적 가정만, 예: “전략 1건당 00만 원 가치” 등).
3. **이익 상승**: 시뮬레이션을 통한 약가 최적화 시 예상되는 **매출·이익 시나리오** (가정을 명시한 범위로만).

### 5.3 심리

- **손실 회피 → 이득 프레이밍**: “비용 120만 원”이 아니라 “매달 N시간 절감 + 기회 가치 00만 원”로 인식되게.
- **앵커**: 계산 결과 숫자가 “투자 대비 가치”의 앵커가 됨.
- **인지적 불일치 해소**: “비싸다”는 생각을 “이만큼 돌려받을 수 있다”는 숫자로 보완.

---

## 6. Expert Review — “신뢰로 마지막 걸림 제거”

- **위치**: 구독 플랜 카드 하단 또는 비교표 아래. “신뢰를 더하는 전문가 검수” 옵션 한 줄.
- **설명 (진실만)**: “AI 결과물에 대해 전문가가 검수·보완해 드리는 옵션입니다. (별도 견적)”
- **심리**: 권위 편향, 후광 효과 — “AI + 전문가” 조합으로 품질·신뢰를 강조.

---

## 7. 심리 원칙별 한줄 적용표

| 원칙 | 적용 |
|------|------|
| **앵커링** | 글로벌 컨설팅 비용(사실) + 원타임 10만·100만·구독 50/120만 노출 순서 |
| **손실 회피 → 이득** | ROI 계산기·플랜 카피를 “절감·이익” 위주로 |
| **Good–Better–Best** | Starter / **Professional** / Enterprise, 중간 강조 |
| **미끼** | Enterprise(고가)로 120만 원이 “합리적 중간”으로 보이게 |
| **호혜** | 무료 견적·데모 먼저 제안 |
| **탈출 경로** | “상담·견적 후 진행은 귀사 결정, 의무 없음” 명시 |
| **노력 인식** | “AI가 방대한 데이터를 처리한 결과”를 시각적으로 보여 줌 (Starter 가치 증명) |
| **후광** | Professional “글로벌 데이터·시뮬레이션 리포트 = 전문성” 강조 |
| **인식 우선** | 드롭다운·비교표로 선택 내용·가격을 다시 노출 |
| **점진적 노출** | 비교표: 모듈 → 크레딧 → 지원 → 가격 순 |
| **넛지** | CTA는 “이 플랜 문의하기” 등 명확하게, 옆에 “의무 없음” |

---

## 8. 금지 사항 (거짓 없이 구매 유도)

- 거짓 결핍 (“남은 N자리”), 거짓 할인, 거짓 사회적 증거 사용 금지.
- ROI·이익 시뮬레이션은 **가정을 명시**하고, 보수적으로만 제시.
- “가장 많이 선택” 등 뱃지는 **실제 데이터** 있을 때만 사용.

---

## 9. 구현 체크리스트

- [x] 원타임: 드롭다운 + 단품 가격(10/100/500/1,000만원) + 견적 문의 연결
- [x] 원타임 묶음: 복수 선택 + 합산 + 견적 문의
- [x] 구독: Starter 50만원 10C, Professional 120만원 40C, Enterprise 문의
- [x] Professional: 뱃지 “성공 확률 극대화”, 시각 강조
- [x] ROI 계산기: 입력 3종 → 출력 3종(시간 절감, 인건비 절감, 전략 가치), 가정 명시
- [x] Expert Review: 한 줄 옵션 + “별도 견적”
- [x] 앵커·호혜·탈출 문구: 상단·원타임 근처·하단에 배치

이 제안서는 **pricing test**의 가격·크레딧·메시지와 **심리 원칙**을 하나로 묶어, “구매할 수밖에 없는” 흐름을 설계한 문서입니다. 모든 문구는 **진실만** 사용하는 전제로 작성되었습니다.
