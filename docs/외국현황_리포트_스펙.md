# 외국 현황 리포트 — 스펙 및 로드맵

## 1. 목표

- **제품/적응증** 기준으로 **주요 국가의 등재·약가·급여 제도**를 한 번에 비교한 리포트를 생성한다.
- 국내 포지셔닝·가격 협상·등재 전략 수립 시 참고 자료로 사용할 수 있는 구조화된 출력을 제공한다.

---

## 2. 입력 (현재)

| 항목 | 설명 | 예시 |
|------|------|------|
| 제품/성분명 | 대상 제품 또는 성분명 | ABC-101, 성분명 |
| 적응증(요약) | 적응증·치료 라인 등 | 2L NSCLC |
| 대상 국가 | 비교할 국가 (쉼표 구분) | 미국, 독일, 일본 |
| 관심 항목 | 약가 수준, 급여 범위, 위험분담제 등 | 약가 수준, RSA 여부 |
| 추가 메모 | 특히 알고 싶은 포인트 | 비교 기준, 시기 |

---

## 3. 출력 (고정)

1. **국가별 등재 여부 및 약가 요약 표**  
   - 국가, 등재여부, 약가수준, 급여범위, 비고
2. **제도/급여 범위 차이**  
   - 국가별 HTA·급여 제도 요약
3. **전략 수립 시 고려 시사점**  
   - 3~5가지 시사점
4. **슬라이드/리포트 초안 구조**  
   - 목차 6단계

(선택) 관심 항목 반영 문구, 추가 메모 섹션

---

## 4. 데이터 소스 (향후)

- **현재**: 데모용 목업(고정 텍스트·표). 제품/적응증/국가 입력은 요약·섹션 문구에만 반영.
- **정식 버전 시 참고할 수 있는 소스**:
  - 미국: FDA, CMS, ASP, 상업보험 정책
  - 독일: G-BA, AMNOG, 약가협상 사례
  - 일본: MHLW, NDB, 약가기준·재평가
  - 영국: NICE, PPRS/VPAG, CDF
  - 프랑스: HAS, ASMR/CT, MEA
  - 기타: IQVIA/공개 리포트, 각국 HTA 기관 공식 문서

---

## 5. API

- **엔드포인트**: `POST /api/ai-module/drug-foreign-landscape`
- **요청**: FormData 또는 JSON. 필드: `product`, `indication`, `countries`, `focus`, `notes`
- **응답**: `{ summary, risks, sections[], assumptions[] }`  
  - `assumptions`: 국가별 등재·약가 요약 표 (배열 of 객체, 테이블 렌더용)

---

| 단계 | 내용 | 상태 |
|------|------|------|

## 7. 화면 (AIModule)

- 경로: `/ai-solutions/drug-foreign-landscape`
- 입력: aiModules 기준 5개 필드
- **산출 기준**: 결과 상단에 세 가지 산출 기준 안내
  - **외국허가**: 주요 국가 등재 여부·허가/급여 기관
  - **약가**: 국가별 약가 수준·협상가·참고가
  - **외국평가현황**: HTA·급여 범위, RSA/MEA 등 평가·제도 현황
- 결과: 요약 → **산출 기준 블록** → **국가별 등재·약가 표**(국가 | 외국허가 | 약가 | 외국평가현황 열 구성) → 제도 차이 → 시사점 → 리포트 구조 → (관심/메모)
- 표는 열 그룹으로 **외국허가(등재여부)**, **약가(약가수준)**, **외국평가현황(급여범위, 비고)** 로 구분해 노출

이 문서는 외국현황 리포트 기능의 공통 스펙·진행 기준으로 사용합니다.
