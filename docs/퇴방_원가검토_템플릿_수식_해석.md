# 퇴장방지 원가검토 template.xlsx 수식 해석

엑셀에서 **하나씩 입력하고 계산해 도출하는** 형태의 템플릿입니다. 시트 2개: **비목계산**, **원가검토가**.

---

## 1. 시트 구성

| 시트 | 역할 |
|------|------|
| **비목계산** | 업소정보·제품정보·연간 제조내역·원료비(원료별)·재료비·노무비·경비·판관비·적정이윤 등 **입력 및 중간 산출** |
| **원가검토가** | 비목계산 결과를 참조해 **업소제출가 vs 원가계산가** 비교, **최종검토가(MIN)**·부가세·유통거래폭·조정신청금액 도출 |

---

## 2. 구분 (국내제조 vs 상품)

- **비목계산!F3**  
  `=IF(OR(LEFT(K4,1)="w",LEFT(K4,1)="E")),"상품","국내제조")`  
  → 제품코드(K4) 첫 글자가 w 또는 E이면 **상품**, 아니면 **국내제조**.

---

## 3. 비목계산 시트 – 입력·산출 구조

### 3.1 원료비 (J~T열)

- **입력**: 원료별  
  - **M** = 구입단가  
  - **N** = 제조투입량  
- **산출**:  
  - **O** = M×N (금액)  
  - **Q31** = 인정총생산량 (= G17, 아래 3.4 참고)  
  - **T7:T30** = O7:O30 / Q31 → **1단위당 원료비(원)**  
  - **T31** = ROUND(SUM(T7:T30), 2) → **1단위당 원료비 합계**  
- **원가검토가 참조**: **BI31** = IF(ISERROR(T31),0,T31) → **원료비(1단위당)** 로 사용.

### 3.2 재료비 (Z~AG열)

- **입력**: 재료별 **Z** = 단가, **AA** = 수량 (또는 유사 구조).  
- **산출**:  
  - **AB** = Z×AA  
  - **AD31** = Q31 (= 인정총생산량)  
  - **AG7:AG30** = AB행 / AD31 → 1단위당 재료비  
  - **AG31** = ROUND(SUM(AG7:AG30), 2) → **1단위당 재료비 합계**  
- **원가검토가 참조**: **BK31** = IF(ISERROR(AG31),0,AG31).

### 3.3 노무비 (AO열)

- **산출**:  
  - **AO31** = ROUND((AO7/AO8*AO9)/AO10, 2)  
  - AO10 = **G17** (인정총생산량)  
  - 즉 (노무 관련 비율·금액) / 생산량 → **1단위당 노무비**.  
- **원가검토가 참조**: **BM31** = IF(ISERROR(AO31),0,AO31).

### 3.4 경비 (BA열)

- **산출**:  
  - **BA31** = ROUND((BA7/BA8*BA9)/BA10, 2)  
  - BA7=AO7, BA8=AO8, BA9=BD13, BA10=**G17**.  
- **원가검토가 참조**:  
  - 국내제조: **BO31** = IF(ISERROR(BA31),0,BA31)  
  - 상품: **BQ17** 등 별도 경비 셀.

### 3.5 인정총생산량 G17, Q31

- **G17** = IF(T4="a", IF(D17>0.03, B17*0.97, C17), IF(OR(T4="b",T4="c"), …))  
  → 투여경로(T4)·손실율(D17) 등에 따라 **인정총생산량** 결정.  
- **Q31** = G17 → 원료비·재료비의 “1단위당” 계산 시 분모로 사용.

### 3.6 제조원가 소계 (BR31)

- **BR31**  
  - 국내제조: BI31+BK31+BM31+BN31+BO31 (원료+재료+노무+외주가공+경비)  
  - 상품: ROUND(BQ18*(1-BR14), 2) 등 상품 전용 산식.

### 3.7 판매 및 일반관리비 (BY31)

- **BY31** = ROUND((BY18*BY20)/BY21, 2)  
  - BY18 = 국내제조/상품 구분에 따른 판관비 산식 (BY17*BY9/BY8 등)  
  - BY20 = BR31 (제조원가 소계)  
  - BY21 = CA22 등.  
- **원가검토가 참조**: **BY31** → 판관비(1단위당).

### 3.8 영업외손익 (CJ31)

- 샘플에서는 **CJ31 = 0** (고정 또는 별도 입력).

### 3.9 적정이윤 (CV31)

- **CV31** = ROUND(((CV8*CV9)/CV10)*CV11, 2)  
  - CV8 = 국내/상품 구분 반영  
  - CV9 = 0.1396 (이윤률 등)  
  - CV10 = CJ16, CV11 = BR31.  
- **원가검토가 참조**: **CV31** → 적정이윤(1단위당).

---

## 4. 원가검토가 시트 – 최종 도출

- **E열** = 업소제출가 (직접 입력 또는 별도 참조)  
- **F열** = **원가계산가** (비목계산 시트에서 참조)  
- **G열** = **최종검토가** = MIN(업소제출가, 원가계산가) → 항목별로 작은 값 채택.

### 4.1 항목별 F열 (원가계산가)

| 행 | 항목 | F열 수식 (원가계산가) |
|----|------|------------------------|
| 7 | 원료비 | IF(국내제조, 비목계산!BI31, 비목계산!BR31) |
| 8 | 재료비 | IF(국내제조, 비목계산!BK31, 0) |
| 9 | 노무비 | IF(국내제조, 비목계산!BM31, 0) |
| 10 | 외주가공비 | 비목계산!BN31 |
| 11 | 경비 | IF(국내제조, 비목계산!BO31, 비목계산!BQ17) |
| 12 | 소계 | SUM(F7:F11) |
| 13 | 판매 및 일반관리비 | 비목계산!BY31 |
| 14 | 영업외손익 | 비목계산!CJ31 |
| 15 | 소계 | F13+F14 |
| 16 | 적정이윤 | 비목계산!CV31 |
| 17 | 계 | F12+F15+F16 |
| 18 | 부가가치세(10%) | ROUND(F17*0.1, 2) |
| 19 | 공장도가 | F17+F18 |
| 20 | 유통거래폭 | (아래 참고) |
| 21 | 총계 | ROUND(F19+F20, 0) |

### 4.2 유통거래폭 (E20, F20, G20)

- **투여경로 E5** = A or C:  
  - **현행상한 E23 < 525** → 공장도가의 **5.15%**  
  - 그 외 → **3.43%**  
- **투여경로 B**:  
  - **E23 < 5257** → 5.15%  
  - 그 외 → 3.43%  
- 수식: `ROUNDDOWN(IF(OR(E5="A",E5="C"), IF(E23<525, E19*5.15/100, E19*3.43/100), IF(E5="B", IF(E23<5257, E19*5.15/100, E19*3.43/100))), 2)`  
  (E19/F19/G19 = 공장도가)

### 4.3 조정신청금액

- **E22** = 비목계산!T3 (업소가 제출한 조정신청금액)  
- **현행 보험상한금액 E23** = 비목계산!Q3  
- 최종 **조정신청금액**은 원가검토가 시트의 **총계(G21)** 등으로 도출.

---

## 5. 웹 구현 시 정리

- **원하는 형태**: 엑셀처럼 **항목별로 하나씩 입력 → 단위당 금액 계산 → 합계 → 원가검토가에서 MIN·부가세·유통거래폭 적용**.
- **비목계산**:  
  - 원료비 = 원료별 (구입단가×투입량) / 인정총생산량 → 합계  
  - 재료비 = 재료별 (단가×수량) / 인정총생산량 → 합계  
  - 노무비·경비 = (해당 비율·총액) / 인정총생산량  
  - 판관비·영업외·적정이윤 = 비목계산 시트의 BY31, CJ31, CV31과 동일 로직으로 1단위당 산출.
- **원가검토가**:  
  - 업소제출가(E) vs 원가계산가(F) → G=MIN(E,F)  
  - 소계 → 부가세 10% → 공장도가 → **유통거래폭(투여경로·현행상한에 따라 5.15% or 3.43%)** → ROUND 총계 = 조정신청금액 후보.

이 해석을 기준으로 웹에서 “템플릿과 동일한 입력 단계 + 수식”을 재현하면 됩니다.
