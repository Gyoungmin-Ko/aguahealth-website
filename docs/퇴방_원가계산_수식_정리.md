# 퇴방 원가계산 수식 정리

- **원가산정방식**: 최종보고서 p.113(표 59)·p.115(표 60) 기준.
- **원가 도출 수식**: 모비졸로정1밀리그램 원가 도출 엑셀.xlsx 추출 결과.

**상태**: 임시 사용 가능(초안). 웹 구현·검증용으로 활용 후, 아래 보완 사항을 반영해 완성도 높일 예정.

---

## 1. 시트 구성

| 시트 | 내용 |
|------|------|
| 1. 원료비 | 원료별 배치투입량·단가·생산배치수 → 1정당 원료비 |
| 2. 재료비 | 재료별 단가·총소요량·총생산량 → 1정당 재료비 |
| 3. 노무비 | 신청제품 노무시간/총생산 노무시간 × 노무비 / 총생산량 → 1정당 노무비 |
| 4. 외주가공비(없음) | (비어 있음, 0 처리) |
| 5. 제조경비 | (제조경비총액 − 연구개발비 − 무형자산상각비) × 비율 / 총생산량 → 1정당 제조경비 |
| 6. 판매 및 일반관리비 | 제품매출 관련 판관비 × 신청제품 제조원가계 / 총 제조원가 → 1정당 판관비 |
| 7. 영업외 손익 | (영업외 수익 − 영업외 비용) 비율 × 신청제품 제조원가계 / 총 제조원가 → 1정당 영업외손익 |
| 8. 적정이윤 | 자기자본세전순이익률 최근 5개년 평균(%) |
| 최종 | 위 비목 합산 → 총원가 → 적정이윤 → 부가세 → 유통거래폭 → 조정신청금액 |

## 2. 시트별 수식 (규격단위당 = 1정당)

### 2.1 1. 원료비
- **연간총생산량(정)** G4 = 13,510,500 (공통)
- **1정당 금액** E = (배치별투입량 × 단가 × 생산배치수) / 연간총생산량  
  - E4 = B4×C4×D4/G4 (G4는 연간총생산량)
- **원료비 합계** E10 = SUM(E4:E9)

### 2.2 2. 재료비
- **1정당 금액** E = 단가 × 총소요량 / 총생산량 (재료별)
- **재료비 합계** E10 = SUM(E4:E9)

### 2.3 3. 노무비
- **1정당 노무비** E4 = (신청제품 노무시간 / 총생산 노무시간) × 노무비 / 총생산량  
  - 즉 (A4/B4)*C4/D4

### 2.4 4. 외주가공비(없음)
- 0 (시트 비어 있음)

### 2.5 5. 제조경비
- **제조경비(연구개발비·무형자산상각비 제외)** C4 = 제조경비총액 − 연구개발비 − 무형자산상각비  
  - B15=제조경비총액, B16=연구개발비, B17=무형자산상각비 → C4 = B15−B16−B17
- **1정당 제조경비** E4 = (신청제품 노무시간 / 총생산 노무시간) × C4 / 총생산량  
  - A4/B4*C4/D4

### 2.6 6. 판매 및 일반관리비
- **제품매출 관련 판관비** A4 = (B11−E11−G11−I11−K11)*(D11/A11)  
  - 손익계산서 수치로부터 산출 (총매출원가·판관비·제조원가·제품매출원가·광고선전비·연구개발비·무형자산상각비·접대비 등)
- **신청제품 제조원가계** B4 = 최종!D14 (제조원가 소계)
- **총 제조원가** C4 = (C11+D11)/2
- **1정당 판매 및 일반관리비** D4 = A4 × B4 / C4

### 2.7 7. 영업외 손익
- **제품매출 관련 영업외손익** A4 = (B15−C15)*D11/A11  
  - 영업외 수익 B15, 영업외 비용 C15, 손익계산서 비율 반영
- **1정당 영업외 손익** D4 = A4 × B4 / C4 (B4=최종!D14, C4=총 제조원가)

### 2.8 8. 적정이윤
- **적정이윤률(%)** = 자기자본세전순이익률 최근 5개년 평균  
  - F5 = AVERAGE(A5:E5) → 예: 7.92, 5.97, 13.36, 15.4, 10.91 → 평균 약 10.712%

### 2.9 최종 (원가계산서)
- **제조원가**  
  - 원료비 D9 = '1. 원료비'!E10  
  - 재료비 D10 = '2. 재료비'!E10  
  - 노무비 D11 = '3. 노무비'!E4  
  - 외주가공비 D12 = 0 (또는 시트4 참조)  
  - 제조경비 D13 = '5. 제조경비'!E4  
  - **소계 D14** = SUM(D9:D13)
- **판매 및 일반관리비** D15 = '6. 판매 및 일반관리비'!D4  
- **영업외 손익** D16 = '7. 영업외 손익'!D4  
- **소계 D17** = SUM(D14:D16)
- **적정이윤** D18 = D17 × C18 (C18 = 0.10712)
- **계** D19 = D17 + D18
- **부가가치세(10%)** D20 = D19 × 0.1
- **유통거래폭(5.15%)** D21 = D19 × 0.0515
- **조정신청금액** D22 = D19 + D20 + D21  
  - 즉 **조정신청금액 = (제조+판관+영업외 소계 + 적정이윤) × (1 + 10% + 5.15%)**
- **현행 보험상한금액** D23 = 참고용 (예: 70원)

## 3. 웹 구현 시 입력·산출 정리

### 3.1 규격단위당(1정당) 직접 입력 가능 항목
- 연간총생산량(정)
- 원료비: 원료별 (배치별투입량, 단가, 생산배치수) 또는 1정당 원료비 합계
- 재료비: 재료별 (단가, 총소요량, 총생산량) 또는 1정당 재료비 합계
- 노무비: 신청제품 노무시간, 총생산 노무시간, 노무비, 총생산량
- 제조경비: 제조경비총액, 연구개발비, 무형자산상각비, (신청제품/총) 노무시간, 총생산량
- 판관비·영업외: 손익계산서 기반 수치 또는 “1정당 금액” 직접 입력 옵션
- 적정이윤률: 5개년 평균 또는 고정값(예: 10.712%)
- 부가가치세: 10%
- 유통거래폭: 5.15% (또는 표 59·60 기준값)

### 3.2 산출
- **조정신청금액(규격단위당)** = (제조원가 소계 + 판관비 + 영업외손익) × (1 + 적정이윤률) × (1 + 10% + 유통거래폭)
