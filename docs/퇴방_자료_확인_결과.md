# 퇴방(퇴장방지의약품) 원가보전 자료 확인 결과

제공해 주신 파일들 확인 결과와, 웹 구현을 위해 추가로 있으면 좋은 것을 정리했습니다.

---

## 1. 읽힌 자료 (구조 파악 가능)

### 1.1 [별지 6] 원가계산서 (PDF) ✅
- **용도**: 최종 원가계산 결과를 담는 서식.
- **확인된 항목**  
  - 상단: 제품명, 업소명, 제품코드, 규격단위  
  - 1Batch당 기준생산량, 1Batch당 실질생산량  
  - **구분별 금액(규격단위당)**  
    - **총원가**  
      - 제조(매입)원가: 원료비, 재료비, 노무비, 외주가공비, 경비, 소계  
      - 판매 및 일반관리비, 영업외 손익, 소계  
      - 적정이윤  
      - 계  
    - 부가가치세(10%)  
    - 유통거래폭  
    - 조정신청금액  
    - 현행 보험상한금액  
  - 확인자/작성자  
→ 웹에서 “원가계산서” 출력 시 이 구조로 매핑하면 됨.

### 1.2 [별지 4] 제품별 상세내역서 (PDF) ✅
- **용도**: 국내제조 제품의 원가 산출을 위한 **입력/증빙** 서식.
- **확인된 섹션**  
  1. 업소 정보 (중소기업 여부, 자산총계, 자기자본, 매출액, 경상이익, 당기순이익 등)  
  2. 제품 제조 정보 (제품명, 약가코드, 외주가공 여부, 생산/재고/출고 수량, 연간생산량, 최종생산시기 등)  
  3. 연간 제조 Batch 내역 (Batch번호, 생산연월, 제조투입량, 실생산량, 손실률, 노무시간, 투입인원, 연간 합계 등)  
  4. 원료비 (원료명, 기초재고, 당기구매, 평균단가, 원료구매내역서)  
  5. 재료비 (재료명, 기초/당기, 평균단가, 재료구매내역서)  
  6. 노무비 (연간 노무시간, 노무비 해당액, 단위 노무공수당 노무비 등)  
  7. 외주가공 내역  
  8. 제조경비 (제조원가명세서상 경비, 연구개발비, 무형자산상각비, 외주가공비 등)  
  9. 판매 및 일반관리비 (손익계산서 기준, 광고선전비, 연구개발비, 접대비 등)  
  10. 영업외 손익 (영업외 수익/비용, 투자활동 관련 명세)  
- 공통 제출자료 안내(결산서, 세무조정계산서, 인력현황 등)도 기재됨.  
→ 웹 “자료입력” 화면은 위 섹션/항목을 단계별로 넣을 수 있게 설계하면 됨.

### 1.3 [별지 5] 상품(수입의약품)별 상세내역서 (PDF) ✅
- **용도**: 수입의약품의 원가 산출을 위한 입력/증빙 서식.
- **확인된 섹션**  
  - 업소 정보  
  - 상품 구매 관련 정보 (제품명, 약가코드, 재고/구매/출고, 연간생산량 등)  
  - 상품 구매 단가, 상품구매 내역서  
  - 판매 및 일반관리비, 영업외 손익 등  
→ 국내제조와 구분해 “수입의약품” 전용 입력 플로우로 설계 가능.

### 1.4 [별지 7] 노무시간 및 인원현황 (PDF) ✅
- **용도**: 노무시간·인원 집계.
- **확인된 항목**: 구분(원료, 고형제, 주사제, 시럽제, 포장작업 등), 노무시간(월평균/연간), 노무인원(월평균/연간).  
→ 제품별 상세내역서의 “노무비” 산출 시 연동 가능.

### 1.5 「필수의약품 안정적 공급을 위한 관리체계 연구」 최종보고서 1부 (PDF) ✅
- **목차 기준**  
  - **표 59** 국내제조 제품 원가산정방식 및 적용기준 → **p.113**  
  - **표 60** 상품(수입의약품) 원가산정방식 및 적용기준 → **p.115**  
  - 퇴장방지의약품 원가보전 분석 등 관련 표 다수 (p.122~ 등).  
- 본문 텍스트 검색으로는 “원가산정방식” 표 내용이 바로 추출되지는 않음.  
→ **p.113~121(표 59·60 포함)** 부분을 직접 보시고, “이 공식/비율로 웹에서 계산한다”고 정리해 주시거나, 해당 페이지만 텍스트로 옮겨 주시면 웹 계산 로직에 반영 가능.

## 2. 읽지 못한 자료 (추가 협의 필요)

### 2.1 퇴장방지의약품 원가산정방식.docx
- **이유**: .docx는 바이너리라 여기서는 내용을 직접 읽을 수 없음.
- **요청**  
  - **방법 A**: “원가산정방식” 요약(산정 공식, 비율, 적용기준)을 **본문 일부를 복사해 텍스트로** 붙여 주시거나,  
  - **방법 B**: 동일 내용을 **.txt 또는 .md로 저장**해 주시면, 그걸 기준으로 서식·계산 로직을 맞추겠습니다.  
  - 최종보고서 **p.113~121**에 동일/유사 내용이 있다면, “그 표 기준으로 한다”고만 알려 주셔도 됨.

### 2.2 퇴장방지검토(10년 3월).xls
- **이유**: .xls는 바이너리라 셀/수식을 직접 읽을 수 없음.
- **용도** (말씀하신 대로): **비목계산**, **원가검토가** 참고.
- **요청**  
  - **방법 A**: “비목계산” 시트와 “원가검토가” 시트에서  
    - **입력 셀**이 무엇인지(예: 원료비, 재료비, 노무비, …)  
    - **최종 산출 셀**이 무엇인지(조정신청금액 등)  
    - **수식 구조**를 간단히(예: “A+B+C”, “소계×10%” 등) 텍스트로 정리해 주시거나,  
  - **방법 B**: 해당 시트를 **CSV로 저장**해 주시면, 열 구조와 숫자 관계를 보고 계산 로직을 설계하겠습니다.  
  - **방법 C**: 수식이 포함된 시트를 **스크린샷**으로 보내 주셔도, 수식/구조를 추려서 반영할 수 있습니다.

## 3. 웹 구현 방향 (현재까지 정리)

| 구분 | 내용 |
|------|------|
| **서식** | [별지 4·5·6·7] PDF 구조를 기준으로 “단계별 입력 화면” 설계 가능. |
| **자료입력** | 국내제조([별지 4]) / 수입([별지 5]) 구분, 원가계산서([별지 6]) 항목과 매핑. |
| **원가계산** | .docx 또는 최종보고서 p.113~121의 **원가산정방식·적용기준** + 엑셀 **비목계산/원가검토가** 수식을 조합해, 동일한 계산이 웹에서 나오도록 구현. |

## 4. 다음에 있으면 좋은 것 (요약)

1. **원가산정방식**  
   - .docx에서 “원가산정방식” 부분을 **텍스트로** 붙여 주시거나,  
   - 최종보고서 **p.113~121**을 “이 표대로 계산한다”고 지정해 주시기.
2. **원가계산 수식**  
   - `퇴장방지검토(10년 3월).xls`의 **비목계산**·**원가검토가** 시트의  
     - 입력 항목  
     - 산출 순서/수식(또는 CSV/스크린샷)  
   → 이걸 기준으로 웹에서 동일 계산 구현.

이 두 가지가 정리되면,  
- “서식·자료입력”은 [별지 4·5·6·7] 구조로,  
- “원가계산”은 위 방식·수식으로  
바로 설계·구현 단계로 넘어갈 수 있습니다.
